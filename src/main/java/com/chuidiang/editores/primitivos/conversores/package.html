Clases de conversión de String a un tipo de dato y de ese tipo de dato a String.

Todas las clases de este paquete implementan la interface InterfaceConversorString. La clase Editor usa esta interface para convertir el texto escrito por el usuario a un tipo concreto de java, como puede ser Double, Integer, etc

Pueden construirse nuevos editores símplemente haciendo nuevas implementaciones de InterfaceConversorString y pasándoselas a la clase Editor
