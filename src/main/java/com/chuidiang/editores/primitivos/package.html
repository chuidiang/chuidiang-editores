Editores de tipos primitivos de Java, como Number, Date, String, etc.

La clase básica de este paquete es Editor. Esta clase admite tres interfaces: InterfaceConversorString, InterfaceTratamientoError e InterfaceRestriccion. El primero es el encargado de convertir el String escrito en el editor a uno de los tipos de datos de Java. El segundo indica al editor cómo mostrar un posible error que haya cometido el usuario al teclear. El tercero verifica que el dato Java cumple ciertas restriccion, como por ejemplo estar entre un valor mínimo y máximo.

Usando esa clase Editor e implementaciones concretas de las tres interfaces en los subpaquetes conversores, errores y restricciones se construyen varios editores concretos, como EditorDate para fecha/hora, EditorLatitud para latitudes geográficas, etc.
