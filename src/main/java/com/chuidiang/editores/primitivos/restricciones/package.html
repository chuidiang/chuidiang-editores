Clases que imponen algún tipo de restricción a los posibles valores que puede meter el usuario en un editor.

Las clases en este paquete implementan la interface InterfaceRestriccion que simplemente tiene un método al que se le pasa el valor recogido del editor y devuelve true o false según dicho valor cumpla o no con la restricción

Están implementadas las siguientes restricciones:

    • RestriccionLongitudString : Aplicable a editores de String. La cadena pasa la restricción si es de una longitud menor que un número dado.
      RestriccionMascaraString : Aplicable a editores de String. La cadena pasa la restricción si es cumple un determinado patrón, como por ejemplo \.*hola\.* (expresión regex)
      RestriccionObjectEquals : Aplicable a cualquier editor. El valor pasa si es equals() a otro.
      RestriccionRango : Aplicable a editores numéricos. El número pasa la restricción si está entre un mínimo y un máximo.
